Detailed Description

Description of a DOS bootsector (plus various common values).

Also see: FAT File System

Note
Because this datastructure has some unusual internal alignments between datafields your compiler must support the non-standard 'packed' attribute. Otherwise change them to byte ...X for the proper values of X.
