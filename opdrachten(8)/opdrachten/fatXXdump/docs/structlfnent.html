Detailed Description

The description of a Long_File_Name fake directory entry.

To encode long file names a sequence of LFN fake directory entries is followed by a normal 8+3 DOS entry for the remainder of that file's information.

For instance the name: "File with very long filename.ext" is encoded as:

seqno field lfn_part (UTF16 characters)
0x03 | 0x40 "me.ext"
0x02 "y long filena"
0x01 "File with ver"
??? A normal 8.3 entry
