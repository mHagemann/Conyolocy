Note:
The notation xxx(N) used in various places means: The unix manual about xxx in manual section N.
For instance: N=2: system calls, N=3: common library functions, N=7: conventions.
The unix manuals usually tell which #include files you need for programs writen in C. When programming in C++ there are often C++ alternatives available, e.g. cstdlib rather then stdlib.h. In this assignment you may use either variant.
