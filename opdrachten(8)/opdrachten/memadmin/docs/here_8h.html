Macro Definition Documentation

#define __HERE__   makeWhere(__FILE__,__LINE__,__PRETTY_FUNCTION__)

The __HERE__ macro produces a "file[line] in function" string that you can use in error messages.

Note
It relies on the __FILE__, __LINE__ and __PRETTY_FUNCTION__ compiler magic names to actually identify lines of source code.
If your compiler does not provide the __PRETTY_FUNCTION__ magic name, you can #define it as __FUNCTION__ (gcc) or __func__ (any compiler). You can also #define it as 0 in which case HERE will omit the name of the function.
